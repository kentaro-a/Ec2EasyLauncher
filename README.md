# Ec2EasyLauncher
Start and Stop your existing EC2 instance.  
  

# Requirements  
- python (3.5.0 or later.)  
- boto3 (1.4.7 or later.)  
```
$ pip install boto3
```  
- awscli (1.11.178 or later.)  
```
$ pip install awscli  
$ aws configure  
  Set your Accesskey, Secretkey, region or stuff like that.
```
